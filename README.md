# Tickitz Movie
Tickitz Moviez is an backend API for moviez-ticketing system built on Express.js

## Installation
1. Clone the repository
```bash
git clone https://github.com/crownss/tickitz-moviez
```
2. Install dependencies
```bash
yarn
```
3. Run the app
```bash
yarn start
```
:fire: You're all set!
